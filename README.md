ATM Project - Core Java

 Description:
This is a simple console-based ATM project developed using *Core Java*.

 Features:
- Login using PIN
- Deposit & Withdraw money
- Check balance
- Mini statement (last 5 transactions)

 Tech Stack:
- Java (Core)
- Scanner, Arrays

 How to Run:
- Open in Eclipse or any IDE
- Run ATMApp.java (main file)
